# WildWish.org Frontend Example with React and Google Materials-UI


## How to use

Clone the [repo](https://github.com/bigcatplichta/wildwish-react-mui):

```sh
git clone git@github.com:bigcatplichta/wildwish-react-mui.git
```

Install it and run:

```sh
npm install
npm start
```
